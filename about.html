About DevPulse

DevPulse is a developer productivity and repository management system built to help software teams track their work more effectively. It brings together commit activity, pull request management, issue tracking and analytics into a single platform.

The system is designed for three types of users — Admins, Managers and Developers. Each role has its own set of permissions and views so that the right people see the right information. Admins can manage users and view system statistics, Managers can approve or reject pull requests, and Developers can log their work and track progress.

The backend is built with Spring Boot and uses JWT based authentication to keep all API endpoints secure. The database is MySQL and all data is stored and managed using Spring Data JPA. The system follows a clean layered architecture with separate entity, repository, service and controller packages.

Technology Stack

Backend: Java 17, Spring Boot 3, Spring Security, JWT (HS512), MySQL, Spring Data JPA, Maven

Frontend: HTML, CSS, JavaScript (Vanilla)

Tools: Postman (API Testing), MySQL Workbench, VS Code

Project Features

✅ User registration and login with role assignment
✅ JWT token generation and validation
✅ Repository linking for GitHub and GitLab
✅ Commit activity logging with auto-generated commit hash
✅ Pull request creation, merge and reject workflow
✅ Issue tracking with status and priority levels
✅ Developer productivity score calculation
✅ Admin panel with system stats
✅ Analytics reports for team, repository and activity
